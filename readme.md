# Como usar? 
- clonar repositorio
- executar comando npm install
## Modo de desenvolvimento 
 `npm run dev`
 ## Modo de produção
 `npm run build`

 `npm start`
## Executar testes 
`npm run test`
## Utilizar sotorybook

`npm run storybook`
acessar a porta 6006
# Criar components
```bash
  npx generate-react-cli component NOME_DO_COMPONENTE
```
# Criar pages
```bash
  npx generate-react-cli component HomePage --type=page
```
